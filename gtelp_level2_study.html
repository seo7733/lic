📝 문법 (Grammar)
26문항 · 20분 · 4지선다 — 가장 단시간에 점수 올릴 수 있는 영역

1. 시제 (Tense) ★★★ 최빈출

G-TELP 문법의 핵심. 특히 완료시제 / 진행시제 구분이 매회 출제됩니다.

현재완료 vs 과거
She _____ in Seoul for 10 years.
(그녀는 10년째 서울에 살고 있다)
→ has lived
for + 기간 = 현재완료. "lived"(과거)는 지금도 살고 있음을 표현 못함
과거완료 vs 단순과거
By the time he arrived, she _____ already left.
→ had
by the time + 과거 = 과거완료(had p.p.) 사용

2. 가정법 (Subjunctive) ★★★ 최빈출

가정법 과거(현재 사실 반대), 가정법 과거완료(과거 사실 반대)를 반드시 구분해야 합니다.

가정법 과거
If I _____ rich, I would travel.
(지금 부자라면 여행 갈 텐데)
→ were
If + were/동사과거, would + 동사원형
가정법 과거완료
If he had studied, he _____ passed.
→ would have
If + had p.p., would have + p.p.

3. 조동사 (Modal Verbs) ★★☆ 빈출

must / should / can / may / would 의 뉘앙스 차이와 완료형 조합을 확인하세요.

He must _____ forgotten the meeting. (그는 분명히 회의를 잊었을 것이다)
→ have (must have forgotten)
과거 추측: must have + p.p. / should have + p.p. (했어야 했는데) 구분 중요

4. 관계절 (Relative Clauses) ★★☆ 빈출

who / which / that / whose / where / when 선택과 계속적 용법(콤마) 여부를 구분합니다.

The firefighter _____ rescued the child received an award.
→ who
사람 선행사 → who/that. 콤마 없으면 제한적 용법 → that 가능

5. 준동사 — to부정사 / 동명사 ★★☆ 빈출

특정 동사가 to부정사만, 또는 동명사만 목적어로 받는지 암기가 필요합니다.

to부정사만 받는 동사
want, plan, hope, decide,
expect, refuse, promise
동명사만 받는 동사
enjoy, finish, mind, avoid,
deny, consider, suggest

6. 수동태 / 능동태 ★☆☆ 기본

The report _____ by the team last week.
→ was written
주어가 행동의 대상 → 수동태 be + p.p. / 시제는 last week → 과거

문법 — 핵심 7단원 집중

  • 시제·가정법 완전 정복 (매회 6~8문항)
  • 조동사 완료형 패턴 암기
  • 관계절 who/which/whose 구분
  • 준동사 목적어 동사 목록 암기
  • 20분 내 전 문항 풀고 검토 시간 확보

청취 — 선독과 메모 습관화

  • 음원 전 10초, 보기 먼저 읽기
  • 숫자·이름·장소 들릴 때 즉시 메모
  • but/however 이후 내용이 핵심
  • paraphrase된 선택지 훈련
  • Part 4 강의형: 주제 파악 후 세부 정보

독해 — 스캔리딩 + 시간 분배

  • 지문당 13분 초과 금지
  • 문제 먼저 읽고 키워드 찾기
  • 어휘 문제: 문맥 대입법 사용
  • 추론 문제: 극단적 보기 먼저 제거
  • 모르는 단어는 건너뛰고 문맥으로 추론

D-30일 학습 플랜

  • 1~10일: 문법 핵심 이론 + 기출 100문항
  • 11~20일: 청취 파트별 실전 연습
  • 21~25일: 독해 지문 + 고빈도 어휘
  • 26~29일: 모의고사 3회 + 오답 분석
  • D-1: 컨디션 관리, 가정법 최종 확인
